## Example

### 邮件分类

假设我们有一张统计表，标识了一些邮件的分类情况。邮件如果包含某特征为1，反之为0。设计算法需要判断之后的邮件是否为垃圾邮件。例如特征为[0,1,1,0]或者[1,1,1,1]的邮件。

| 序号 | feature 1 | feature 2 | feature 3 | feature 4 | Is it spam |
| :--: | :-------: | :-------: | :-------: | :-------: | :--------: |
|  1   |     1     |     1     |     0     |     0     |     0      |
|  2   |     1     |     0     |     1     |     1     |     0      |
|  3   |     0     |     0     |     1     |     1     |     1      |
|  4   |     1     |     0     |     0     |     0     |     0      |
|  5   |     0     |     0     |     1     |     1     |     1      |
|  6   |     1     |     0     |     0     |     0     |     0      |
|  7   |     0     |     1     |     1     |     1     |     1      |
|  8   |     0     |     1     |     0     |     0     |     0      |
|  9   |     1     |     0     |     0     |     1     |     0      |
|  10  |     0     |     1     |     1     |     0     |     1      |

该问题是一个典型的分类问题，可以使用朴素贝叶斯分类器判断具有以上特征的邮件是否为垃圾邮件
